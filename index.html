SIRV Configurator (rotoscoper)

Use this tool to create hotspot coordinates over a series of frames.

Enter the URL of your spin file in the form below and fetch. Your complete spin file load in the text area below. Enter your desired configuration values (style and box coordinates) into the appropriate fields. Scroll through each image and click where you want the hotspot to be located. Your hotspot values can be found in the New Config text area. This tool will only handle one hotspot at a time. You you will have to copy out the new config and add it to your file manually.
